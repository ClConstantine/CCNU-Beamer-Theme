# CCNU-Beamer-Theme

## Overview

`CCNU`风格的`Beamer`主题，可用于答辩、汇报、~~装逼~~等.

使用校官网`RGB(57, 130, 151)`作为底色.

使用方式和详细细节见源代码和`slide`.
